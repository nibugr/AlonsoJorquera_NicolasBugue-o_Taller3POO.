##  Objetivos del Taller
- Diseñar e implementar un sistema de gestión de proyectos
- Aplicar patrones de diseño (Singleton, Factory, Strategy, Visitor)
- Implementar persistencia en archivos de texto
- Desarrollar menús interactivos para diferentes roles de usuario

## Patrones de Diseño Implementados
- **Singleton**: GestorArchivos, Sistema, servicios
- **Factory**: Creación de tareas (Bug, Feature, Documentación)
- **Strategy**: Estrategias de priorización de tareas
- **Visitor**: Acciones específicas por tipo de tarea
